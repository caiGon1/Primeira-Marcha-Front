# 🚗 Plataforma de Agendamento de Aulas de Direção

Uma **webaplicação desenvolvida com React**, utilizando **TailwindCSS** para estilização e **Material UI (MUI)** para componentes de interface. O projeto tem como objetivo **conectar alunos a instrutores independentes de direção**, facilitando o agendamento de aulas práticas de forma simples e digital.

## 📌 Contexto

Com as recentes mudanças no processo de obtenção da **primeira habilitação**, tornou-se possível realizar aulas práticas com **instrutores independentes**, sem a obrigatoriedade de intermediação exclusiva de autoescolas.

Esta plataforma foi criada para **facilitar esse novo modelo**, conectando **alunos (condutores)** e **instrutores**, permitindo que encontrem horários disponíveis, comparem preços e agendem aulas diretamente pela aplicação.

## ⚙️ Tecnologias Utilizadas

* **React** – construção da interface da aplicação
* **TailwindCSS** – estilização e layout responsivo
* **Material UI (MUI)** – componentes de interface modernos
* **JavaScript / Node.js** – lógica da aplicação

## ✨ Funcionalidades

### 👤 Para Condutores (Alunos)

* Cadastro e login na plataforma
* Busca por **instrutores disponíveis por data e horário**
* Visualização de **preço da aula e localização do instrutor**
* **Agendamento de aulas práticas**
* Pagamento diretamente pela plataforma
* Sistema de **mensagens com o instrutor**

### 👨‍🏫 Para Instrutores

* Cadastro e gerenciamento de perfil
* Definição de **disponibilidade de horários**
* Definição do **valor da aula**
* Recebimento e gerenciamento de agendamentos
* Comunicação direta com os alunos

### 💬 Comunicação

A aplicação possui um sistema de **mensagens internas**, permitindo que **instrutores e condutores conversem diretamente** para alinhar detalhes da aula.

## 🎯 Objetivo do Projeto

O objetivo da plataforma é **simplificar o processo de encontrar e agendar aulas de direção**, tornando o sistema mais acessível e eficiente para alunos e instrutores independentes.
